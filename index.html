Solar Prominence Plasma Flow Analysis

Analysis of UV data from the EUI imager aboard ESA's Solar Orbiter probe. Developing Python-based pipelines to characterize plasma flows in solar prominences through time-series analysis of intensity variations.

Key Objectives

  • • Quantify energy transport in prominence structures

Developed a Python pipeline for detecting and analyzing exoplanet transits using photometric data. Applied Box Least Squares (BLS) algorithm to identify periodic dips in stellar brightness.

Achievements

Machine learning approach to stellar classification using spectroscopic data. Implemented feature extraction from absorption lines to determine stellar temperature, metallicity, and surface gravity.
